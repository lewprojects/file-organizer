# File_Organizer

## What is this?

This is a file organizer was made based on the way I personally like to organize a bunch of messy files before I sort them out.

## Requirements

You must have python3 installed.

## How do you run it?

1. Download the file.
2. In terminal, from your downloads directory, type python3 file_organizer.py

## What do I need to give it?

After running the program, you will be prompted to paste in the folder path you want to organize. Copy and paste it in.

## What does the output look like?

The files will be put into folders according to the year they were last modified, then into subfolders according to their file type. If files with duplicate names are found, the program will automatically adjust the file name to file (1).txt, file (2).txt, etc.
